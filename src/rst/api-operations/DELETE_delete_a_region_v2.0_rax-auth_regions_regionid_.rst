Delete A Region
~~~~~~~~~~~~~~~~~~~~~~~~~

`Request <DELETE_delete_a_region_v2.0_rax-auth_regions_regionid_.rst#request>`__
`Response <DELETE_delete_a_region_v2.0_rax-auth_regions_regionid_.rst#response>`__

.. code-block:: javascript

    DELETE /v2.0/RAX-AUTH/regions/{regionId}

Deletes a region.

This request only succeeds if the region default is set to false. The default value must be false before this operation will be allowed.
